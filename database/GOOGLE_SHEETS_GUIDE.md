# ฟอร์ม Google Sheet สำหรับนำเข้าข้อมูล — UNIPHARMA

ระบบนำเข้าได้ **2 ชนิด**: **ยา** (รวมสต็อกตั้งต้น) และ **ผู้จัดจำหน่าย**
ทำ Google Sheet 1 ไฟล์ มี 2 แท็บ ตามหัวคอลัมน์ด้านล่าง (ห้ามเปลี่ยนชื่อหัวคอลัมน์ ระบบจะจับให้อัตโนมัติ)

---

## วิธีสร้างฟอร์มเร็วสุด (แนะนำ)
1. สร้าง Google Sheet ใหม่ที่ https://sheets.new
2. **File → Import → Upload** ไฟล์ [`templates/drugs_template.csv`](templates/drugs_template.csv)
   → เลือก **Insert new sheet(s)** → ตั้งชื่อแท็บว่า **ยา**
3. ทำซ้ำกับ [`templates/suppliers_template.csv`](templates/suppliers_template.csv) → ตั้งชื่อแท็บว่า **ผู้จัดจำหน่าย**
4. ลบแถวตัวอย่างออก แล้วกรอกข้อมูลจริงของคุณ

---

## แท็บ 1 — ยา (Drugs)

| หัวคอลัมน์ | ความหมาย | ตัวอย่าง | จำเป็น |

## วิธีนำเข้าเข้าระบบ

### วิธี A — อัปโหลดไฟล์ (ชัวร์สุด ไม่มีปัญหา CORS) ✅ แนะนำ
1. ใน Google Sheet → เลือกแท็บที่ต้องการ → **File → Download → Comma-separated values (.csv)**
   (หรือ Microsoft Excel .xlsx)
2. ในเว็บ UNIPHARMA → เมนู **🔄 ซิงค์ข้อมูล** → แท็บ **อัปโหลดไฟล์**
3. เลือกชนิด (ยา/ผู้จัดจำหน่าย) → ลากไฟล์ลง → ระบบจับคู่คอลัมน์อัตโนมัติ → **Preview → นำเข้าข้อมูล**

### วิธี B — ลิงก์ Google Sheets โดยตรง
1. ใน Google Sheet → **Share → General access → Anyone with the link → Viewer**
2. เปิดแท็บที่ต้องการ คัดลอก URL จากแถบที่อยู่ (URL จะมี `gid=...` ของแท็บนั้น)
3. ในเว็บ → 🔄 ซิงค์ข้อมูล → แท็บ **Google Sheets** → วาง URL → **ดึงข้อมูล** → Preview → นำเข้า
   > หากดึงไม่สำเร็จ (บางองค์กรบล็อก) ให้ใช้วิธี A แทน

## ทุก 2 เดือน (อัปเดตข้อมูล)
แก้ข้อมูลใน Google Sheet เดิม → นำเข้าซ้ำ — ระบบ **Merge** โดยยึด `code`/`id`:
แถวที่รหัสซ้ำ = อัปเดตทับ, รหัสใหม่ = เพิ่มเข้า, **ไม่ลบของเก่า**
เมื่อเชื่อม Supabase แล้ว ข้อมูลจะถูกบันทึกขึ้นคลาวด์ให้อัตโนมัติทุกครั้งที่นำเข้า
